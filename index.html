Helix Agent

A powerful evolutionary AI agent framework built with Hexagonal Architecture.

Vision

Helix Agent is a revolutionary platform for creating increasingly intelligent and cost-effective autonomous agents through digital DNA evolution. Our system allows agents to self-improve, specialize, and collaborate while minimizing LLM usage costs.

Technology Stack

  • Node.js
  • TypeScript
  • Express
  • pnpm
  • Hexagonal Architecture
  • Dependency Injection with tsyringe
  • Logging with pino and chalk

Core Features

Implemented

  • Clean hexagonal architecture
  • Dependency injection for better testability
  • Environment-based configuration
  • Basic agent functionality with REST API

Getting Started

Prerequisites

  • Node.js >= 16
  • pnpm >= 6

Installation

  1. Clone the repository
  2. Install dependencies:
pnpm install
  1. Create a .env file:
PORT=3000
NODE_ENV=development
LOG_LEVEL=info

Development

Run the development server:

pnpm dev

Testing

Run the API tests (while the server is running):

pnpm test:api

Build

Build the project:

pnpm build

Project Structure

src/
├── adapters/             # Adapters layer (controllers, repositories)
│   ├── primary/          # Driving/primary adapters (REST API)
│   └── secondary/        # Driven/secondary adapters (persistence)
├── application/          # Application layer (use cases)
├── config/               # Configuration
├── domain/               # Domain layer (business logic)
│   ├── entities/         # Domain entities
│   ├── ports/            # Ports (interfaces)
│   └── services/         # Domain services
└── utils/                # Utilities

API Endpoints

Current

  • GET /health - Health check
  • POST /api/agents - Create a new agent
  • GET /api/agents/:id - Get agent by ID

Planned

  • POST /api/agents/:id/evolve - Trigger agent evolution
  • POST /api/agents/:id/train - Train agent on specific task
  • POST /api/agents/breed - Create child agent from two parents
  • GET /api/agents/:id/dna - Get agent's DNA structure
  • GET /api/generations - List agent generations
  • POST /api/simulations - Run simulation for training
