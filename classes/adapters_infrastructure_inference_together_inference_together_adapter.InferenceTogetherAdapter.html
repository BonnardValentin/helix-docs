Class InferenceTogetherAdapter

Implementation of the InferenceTogetherPort using Together.ai's API

This adapter provides LLM inference capabilities through Together.ai's API, managing agent registration, inference requests, and result tracking. It integrates with the MCP (Model Control Protocol) for resource and tool management.
