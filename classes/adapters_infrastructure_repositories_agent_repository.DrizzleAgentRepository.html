Class DrizzleAgentRepository

Implementation of the AgentRepositoryPort using Drizzle ORM

This repository handles all database operations related to Agent entities, including CRUD operations and specialized queries for agent capabilities. It uses Drizzle ORM for database interactions and includes proper error handling through the Either type.
